O que são Clients?

(Voltar)

Os clients são os clientes do Modelo Cliente-Servidor e podemos acessar os sites através dos browsers programados com linguagem frontend.

Linguegem frontend são conhecidas por ser linguagem de marcação que estilzam uma interface para o usuário. As mais conhecidas e utilizadas são HTML, CSS, Angular e Javascript.

O que são Servers?

(Voltar)

Servers já são o lado do Servidor do Modelo Cliente-Servidor, em que se localiza todo o banco de dados e hospedagem do site, site esse que é programado com linguagem backend, as mais conhecidas sendo Java e C#, por exemplo.

Como é uma estrutura básica de um HTML?

(Voltar)

São compostas por tags específicas que estruturam o HTML, sendo elas a própria tag html, o head e o body. Todas as tags ficam dentro do html. O head é o cabeçalho e o body é literalmente o corpo do texto do site.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas dentro de uma tag HTML nada mais é do listas numeradas com os tópicos necessários.

Já as listas mão ordenadas, aparecem com um ponto antes do tópico em questão, e não com números, por isso são dadas como não ordenadas, pois não posuem ordem.
